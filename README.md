# 排班日历

智能排班规律管理应用，支持白班、夜班、休息等多种班次类型。

## 功能特性

- 🎯 **排班规律设置** - 灵活配置班次循环规律
- 📅 **日历视图** - 直观查看排班安排
- 🌓 **主题切换** - 支持暗色/亮色主题
- ⏱️ **休息日倒计时** - 显示距离下一个休息日的天数
- 🎂 **重要日期** - 标记生日、纪念日等
- 📤 **导出功能** - 导出图片、JSON 备份、打印
- 📱 **PWA 支持** - 可安装到手机桌面，离线可用

## 使用方法

1. 设置班次类型和排班规律
2. 选择起始日期
3. 点击"生成排班"

## 本地运行

```bash
npx serve .
```

然后访问 http://localhost:3000

## 技术栈

- HTML5 / CSS3 / JavaScript
- PWA (Service Worker)
