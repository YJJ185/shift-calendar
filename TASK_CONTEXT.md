# 排班日历 - 导出功能改进任务

## 📋 快速上下文

**当前问题**: 导出为图片功能效果不好，导出的图片和屏幕显示不一致

**根本原因**: 使用 html2canvas 库截图，但它无法正确处理：
- CSS 变量 `var(--xxx)`
- 伪元素 `::before` / `::after`
- 复杂渐变和阴影

**已确定方案**: 放弃 html2canvas，改用 **Canvas API 原生绘制**
- 完全免费（浏览器原生API）
- 用户已同意先试试看效果
